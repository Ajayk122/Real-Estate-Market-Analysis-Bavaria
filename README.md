# Real-Estate-Market-Analysis-Bavaria
Analyzed Bavaria's rental housing market using large-scale listing data to understand how property characteristics, location, and listing behavior influence rental prices and market dynamics. 
- I cleaned and structured large real estate datasets using Python (Pandas, NumPy)
- Built regression models to identify key drivers of rental price formation
- Applied clustering techniques to uncover market segmentation patterns
- Performed geospatial analysis using Haversine distance — measuring proximity to city centers.
- Analyzed listing text to extract behavioral patterns in the rental market
- Built interactive dashboards to support data-driven decision making.
